Сапер

    Повтор игры

      Справка

      Доступные функции:
      - Новая игра: Позволяет начать новую игру с заданными параметрами (ширина, высота и количество мин).
      - Просмотр прошлых игр: Открывает список всех сохранённых игр. Чтобы посмотреть повтор конкретной игры, нажмите на кнопку "Просмотреть прошлые игры" и выберите интересующую игру из списка.
      - Повтор игры: Показывает пошаговую историю выбранной игры, включая информацию о каждом ходе (координаты и результат).
      - Очистка базы данных: Удаляет все сохранённые игры из базы данных.

      Как играть:
      1. Введите ваше имя в поле "Имя игрока".
      2. Установите параметры игры (ширина, высота и количество мин).
      3. Нажмите кнопку "Новая игра", чтобы начать новую игру.
      4. Щёлкайте по ячейкам на игровом поле, чтобы открыть их. Если вы откроете ячейку с миной, игра закончится.
      5. Цель игры — открыть все ячейки без мин.

      Повтор игры:
      Чтобы просмотреть повтор игры:
      1. Нажмите кнопку "Просмотреть прошлые игры".
      2. В появившемся списке выберите игру, кликнув на неё.
      3. Повтор игры будет отображён в отдельном окне, где вы сможете увидеть все шаги и результат.
